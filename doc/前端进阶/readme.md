# Js方面


### JQuery

### Vue.js

学会简单的使用 Vue.js

之前使用的 Angularjs 1.0 ，上手容易，由于1.0 不再更新，当时架构的不好，有许多小bug。2.0相当于一个全新的框架，上手难度满星，所以不推荐使用2.0

Vue2.0 目前国内最火，国际相对较火，中文文档舒服。

要求：熟练地使用基本功能
