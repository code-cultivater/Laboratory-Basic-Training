# MySql

关系型数据库

1. Mysql：官网下载一个就好啦。
    
    要求：
    - 安装到windows服务
    - 查看有几个表
    
2. 可视化工具：Navicat 
    
    要求：
    - 连接本地数据库
    - 熟悉创建表、主键等等
    - 熟悉用户管理
    
3. 客户端中使用mysql
    -  使用java连接mysql，执行一些插入、查询语句等
    - 教程 http://www.runoob.com/java/java-mysql-connect.html
